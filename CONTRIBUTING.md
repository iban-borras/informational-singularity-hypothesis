# 🤝 Contributing to the HSI Agents Project

Thank you for your interest in contributing to the **HSI Agents** project! This repository is part of an open scientific effort to explore the emergence of order from pure information, based on the *Hypothesis of Informational Singularity (HSI)*.

Whether you come from physics, computer science, AI, mathematics, or the arts, your input is valuable. This guide will help you get started.

---

## ✨ Types of contributions welcome

* Improvements to the Φ generator (Level 0 algorithm)
* New pattern detectors (Pₖ)
* Algorithms to infer emergent rules (ωₖ)
* Additional metrics (compressibility, autocorrelation, multifractality…)
* Visualizations (2D/3D, temporal evolution, heatmaps)
* Translations of the README or documentation
* Proposals to validate HSI in scientific or artistic contexts

---

## 🚀 Getting started

1. **Fork** this repository
2. Create a new branch with a clear name: `feature/fractal-detector` or `fix/phi-metric`
3. Make clear, well-documented changes
4. Submit a pull request with a description of your proposal

---

## 🧪 Best practices

* Follow the style of the existing codebase (`snake_case`, docstrings, modularity)
* Accompany contributions with sample data or basic tests if possible
* If you propose a new metric or agent, include your rationale or a short explanation

---

## 💡 Suggested first contributions

* Create an alternative version of the Φ generator using XOR-based collapse
* Design an agent that detects spatial symmetry via FFT
* Add a visual validator that animates φ-alignment over time
* Build a minimal web interface to track fractal dimension and φ-metrics live
